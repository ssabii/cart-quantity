# 장바구니 수량 선택 (Cart Quantity)

## 제한 시간: 1시간

## 요구사항

### 필수 기능
1. **장바구니 목록 표시**
   - 상품명, 가격, 수량, 소계 표시
   - 총 금액 계산 및 표시

2. **수량 변경**
   - +/- 버튼으로 수량 조절
   - 최소 1개, 최대 99개 제한
   - 직접 입력도 가능

3. **상품 삭제**
   - 개별 상품 삭제
   - 삭제 확인 없이 즉시 삭제

4. **품절 처리**
   - 품절 상품은 수량 변경 불가
   - 품절 상품 시각적 구분
   - 품절 상품은 총 금액에서 제외

### UI 요구사항
- 수량 변경 시 소계/총액 즉시 반영
- 장바구니가 비었을 때 빈 상태 표시
- 품절 상품 명확한 시각적 구분

## 추가 도전 과제 (시간 남으면)
- 수량 변경 시 낙관적 업데이트 + 롤백
- 전체 삭제 기능
- 선택 삭제 기능 (체크박스)
- 금액 3자리마다 콤마 포맷팅

## 시작하기

```bash
npm install
npm run dev
```

